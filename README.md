# 📅 Agendador de Horários (Spring Boot API)

## 📌 Sobre o Projeto

Este projeto consiste em uma API REST desenvolvida com **Spring Boot** para gerenciamento de agendamentos de serviços.

A aplicação permite o cadastro de agendamentos contendo informações como:
- Serviço
- Profissional
- Cliente
- Telefone
- Data e horário

O objetivo do projeto é simular um sistema real de agendamento (como barbearias, clínicas ou prestadores de serviço), aplicando conceitos de **back-end moderno, arquitetura em camadas e boas práticas com Spring**.

---

## 🧠 Objetivo Técnico

O projeto foi desenvolvido com foco em:

- Prática de **API REST**
- Uso de **Spring Boot + JPA**
- Manipulação de datas com `LocalDateTime`
- Tratamento de erros reais (HTTP 400, 500, etc.)
- Estruturação em **camadas (Controller → Service → Repository)**
- Simulação de regras de negócio reais
